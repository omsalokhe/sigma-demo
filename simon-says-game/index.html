How to Play

       

            Welcome to Simon Says! The game is simple:            

                   
  1. Watch the sequence of flashing colors.
    2.                
  2. When the sequence is complete, repeat it by clicking the buttons in the same order.
    3.                
  3. Each time you get it right, the sequence will get longer.
    4.                
  4. If you make a mistake, the game is over.
    5.            
       

           

Game Over!
